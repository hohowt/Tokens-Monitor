# AI Token 监控 — 安装使用指南

## 这是什么

这是公司内部的 AI 编程助手用量监控插件，安装后会自动统计你在 VS Code / Cursor 中使用 Copilot、Claude、Cline、Continue 等 AI 工具产生的 Token 用量，并上报到公司服务器，方便统一管理和成本核算。

**安装后无需任何额外操作，插件在后台自动运行，不影响正常编码。**

---

## 1. 安装

### 第一步：获取安装包

向管理员获取扩展安装包文件（`.vsix` 格式），例如：

```
ai-token-monitor-win32-x64.vsix
```

### 第二步：安装到编辑器

**VS Code：**

1. 打开 VS Code
2. 按 `Ctrl+Shift+P` 打开命令面板
3. 输入 `Install from VSIX` 并回车
4. 选择拿到的 `.vsix` 文件
5. 安装完成后，按提示点击 **「重载窗口」**

**Cursor：**

操作完全一样 — `Ctrl+Shift+P` → `Install from VSIX` → 选文件 → 重载。

> 如果你同时使用 VS Code 和 Cursor，两边各装一次即可，数据会分别上报。

---

## 2. 配置身份信息

安装后首次启动，左侧活动栏会出现监控图标，插件会弹出提示引导你配置身份。

### 方式一：在面板中填写（推荐）

1. 点击左侧活动栏的 **监控图标**
2. 在打开的面板中填写：
   - **用户 ID**（工号或英文用户名）
   - **姓名**（中文姓名）
   - **部门**
3. 填写后自动保存生效

### 方式二：在设置中填写

1. 按 `Ctrl+,` 打开设置
2. 搜索 `aiTokenMonitor`
3. 填写 `User Id`、`User Name`、`Department`

> **重要：** 用户 ID 和姓名必须填写，否则用量数据无法归属到个人。

---

## 3. 日常使用

**什么都不用做。** 配置好身份后，插件全自动运行：

- 左下角状态栏显示当天累计 Token 数
- 点击左侧监控图标可查看详细使用统计
- 所有 AI 请求（Copilot 补全、Chat 对话等）自动被记录

---

## 4. 自动更新

插件内置了自动更新功能：

- 每天打开编辑器时自动检查新版本
- 有更新时会弹出通知：**「AI Token 监控有新版本 vX.X.X」**
- 点击 **「立即更新」** → 自动下载安装 → 按提示重载窗口
- 更新完成，继续使用即可

> 如果暂时不想更新，可以点「跳过此版本」。

---

## 5. 常见问题

### 安装后 AI 工具（Copilot 等）不能用了

点击重载窗口提示后应恢复正常。如果仍有问题：

- 检查公司是否有网络代理（VPN），如有，打开设置搜索 `aiTokenMonitor.upstreamProxy`，填入代理地址（如 `http://proxy.company.com:8080`）

### 状态栏没有显示 Token 数字

检查身份信息是否已填写完整（用户 ID 和姓名缺一不可）。

### 想暂时关闭监控

打开设置，搜索 `aiTokenMonitor.transparentMode`，设为 `false`。需要时改回 `true`。

### 卸载

`Ctrl+Shift+P` → 输入 `Uninstall` → 找到「腾轩旅游集团 - AI Token 监控」→ 卸载 → 重载窗口。
